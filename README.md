# 🔮 清风道长·生肖姓名学分析系统

陈安茂派正统理论 · AI 自动化分析

## 🌟 功能特点

- ✅ 生肖自动计算
- ✅ 五行分析
- ✅ 姓名评分
- ✅ 小名推荐
- ✅ PDF 报告下载

## 🚀 本地运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行网站
streamlit run src/app.py --server.port=8501 --server.address=0.0.0.0
```

## 🌐 在线访问

[Streamlit Cloud](https://share.streamlit.io/)

## 📋 使用说明

1. 输入姓名
2. 选择性别
3. 选择出生日期
4. 点击"开始分析"
5. 查看分析结果
6. 下载 PDF 报告
